Each folder in this directory is described as follows:
ipcore: All IP cores used by the TSN network card example project and their storage paths
rtl: Device-related code and interface-related code of the TSN network card example project
create_project.tcl: Script file for creating TSN network card sample project
Makefile: Makefile for compiling the TSN network card example project. For details, please refer to Chapter 3 of the TSN network card user manual in the OpenTSN3.2-centralized\Hardware\doc\ manual directory
TSN_FPGA_4port.sdc: Design constraints file for the TSN NIC example project
