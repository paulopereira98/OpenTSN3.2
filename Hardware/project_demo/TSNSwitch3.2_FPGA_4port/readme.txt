Each folder in this directory is described as follows:
ipcore: All IP cores used by the TSN switch example project and their storage paths
rtl: Device-related code and interface-related code of the TSN switch example project
create_project.tcl: Script file for creating TSN switch example project
Makefile: Makefile for compiling the TSN switch example project. For details, please refer to Chapter 3 of the TSN Switch User Manual under the OpenTSN3.2-centralized\Hardware\doc\ User Manual directory
TSN_FPGA_4port.sdc: Design constraints file for the TSN switch example project
