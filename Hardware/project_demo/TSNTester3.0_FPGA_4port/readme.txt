Each folder in this directory is described as follows:
ipcore: All IP cores used by the TSN tester example project and their storage paths
rtl: Device-related code and interface-related code of the TSN tester example project
create_project.tcl: script file for creating TSN tester example project
Makefile: Makefile for compiling the TSN tester example project. For details, please refer to the third chapter of the TSN network card manual in the OpenTSN3.0-centralized\Hardware\doc\ manual directory (Note: the operation of the TSN tester Makefile and TSN The operation of the network card Makefile is the same)
TSN_FPGA_4port.sdc: Design constraints file for the TSN tester example project
