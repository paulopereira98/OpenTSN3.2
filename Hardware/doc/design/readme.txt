1. Each file in this directory is described as follows:
Design scheme of TSN network card (TSNNic3.2): This document introduces the functions and interface signals of time-sensitive network side processing logic (TSE) and time-sensitive network hardware control logic (HCP).
TSN Switch (TSNSwitch3.2) Design Scheme: This document describes the functions and interface signals of Time Sensitive Network Switch Processing Logic (TSS) and Time Sensitive Network Hardware Control Logic (HCP).
Design scheme of TSN tester (TSNTest3.0): This document is the design scheme of the hardware control logic of the TSN tester, including its overall structure and message processing flow.
