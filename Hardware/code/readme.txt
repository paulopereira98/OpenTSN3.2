The contents and purposes of each folder in this directory are described as follows:
TSNNic3.2: It mainly includes the source code of the TSN network card 3.2 hardware logic core and the parameter description file used to the IP core
TSNSwitch3.2: It mainly includes the source code of the TSN switch 3.2 hardware logic core and the parameter description file used to the IP core
TSNTester3.0: It mainly includes the source code of TSN tester 3.0 hardware logic core and the parameter description file used to IP core
